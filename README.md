# 🛠️ Local setup

1. Install dependencies

```bash
npm install
```

2. Start local server

```bash
npm run dev
```

> **Note** > <br>Navigate to http://localhost:3333 to see your app

# 📢 Production

1. Install dependencies

```bash
npm install
```

2. Build & start app

```bash
# Navigate to http://localhost:3333 to see your app
npm run prod
```
